# EliteCars
Tienda online de vehículos de lujo EliteCars. Incluye panel administrativo para gestionar usuarios (registro, login, perfiles) y catálogo de vehículos (Porsche GT3 RS, Mercedes, Lamborghini Urus, Mini Cooper, VW Golf, Brabus). Interfaz responsiva con componentes web reutilizables en JavaScript para actualizar dinámicamente precios y disponibilidad.
